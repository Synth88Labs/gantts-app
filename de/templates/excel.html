Gantt-Diagramm-Vorlage für Excel

Eine Gantt-Diagramm-Vorlage für Excel, fertig aufgebaut: Spalten für Vorgang, Start, Ende, Dauer und Fortschritt, dazu das bereits konfigurierte Diagramm. Herunterladen, Daten ersetzen — ohne sich mit unsichtbaren Datenreihen herumzuschlagen.

⬇ Excel (.xlsx) ⬇ PowerPoint ⬇ CSV ✎ Kostenlos online bearbeiten

Was enthalten ist

Die Struktur, die Excel erwartet, bereits mit Beispieldaten gefüllt:

  • Datenblatt — Vorgang, Start, Ende, Dauer und Fertigstellungsgrad im Format, das das Diagramm braucht.
  • Fertiges Diagramm — Gestapelte Balken, Startreihe ohne Füllung, Kategorienachse umgekehrt.
  • Beispielphasen — Vier Phasen mit Vorgängen zum Überschreiben.
  • Meilensteine — Zeilen mit Dauer null für Lieferungen und Freigaben.
Excel zeichnet das Diagramm, versteht aber keine Abhängigkeiten: Verschieben Sie einen Vorgang, merkt kein anderer etwas davon. Für einen kurzen, stabilen Plan reicht das; sobald Vorgänge verknüpft sind, kostet die Pflege von Hand mehr Zeit als die Planung selbst.

So passen Sie sie an

  1. Beispielzeilen durch eigene Vorgänge ersetzen und dabei das Datumsformat beibehalten.
  2. Minimum der Horizontalachse auf Ihr Startdatum setzen, sonst beginnt Excel im Jahr 1900.
  3. Zeilen innerhalb einer Phase ergänzen und den Datenbereich des Diagramms erweitern.
  4. Farbe je Phase statt je Vorgang — deutlich besser lesbar.

Hinweise zur Terminplanung

  • Termine müssen echte Datumswerte sein, kein Text, sonst platziert das Diagramm sie falsch.
  • Löschen Sie die Startreihe nicht. Sie ist unsichtbar, positioniert aber jeden Balken.
  • Brauchen Sie Abhängigkeiten, wechseln Sie das Werkzeug. In Excel nachzubauen ist verlorene Zeit.
  • Zum Teilen als PDF exportieren; wer die .xlsx verschickt, lädt zum Zerlegen des Diagramms ein.

Häufige Fragen

Wie verwende ich die Vorlage in Excel?

Datei herunterladen, Datenblatt öffnen und die Beispielvorgänge samt Terminen ersetzen. Das Diagramm aktualisiert sich selbst, solange das Spaltenformat erhalten bleibt.

Funktioniert sie in älteren Excel-Versionen?

Ja. Sie nutzt ein normales gestapeltes Balkendiagramm, das in jeder modernen Version verfügbar ist — ohne Makros und ohne Add-ins.

Kann ich Abhängigkeiten ergänzen?

In Excel nicht. Wenn Sie sie brauchen, öffnen Sie die CSV im kostenlosen Editor und verknüpfen die Vorgänge dort.

Ist sie kostenlos?

Ja, ohne Anmeldung und ohne Wasserzeichen.
