Automatischer kritischer Pfad

Sehen Sie genau die Aufgabenkette, die Ihren Endtermin bestimmt — hervorgehoben, sobald sich Termine ändern. Ohne Handrechnerei, ohne Upgrade.

Planen per Drag-and-drop

Balken auf der Zeitachse verschieben und skalieren, Fortschritt in Prozent setzen, Aufgaben umsortieren. Der Plan hält mit.

Meilensteine, Phasen und PSP

Fassen Sie Arbeit in Phasen, setzen Sie die Meilensteine, auf die es ankommt, und nummerieren Sie den Projektstrukturplan. Erst die Hierarchie macht einen großen Plan lesbar.

Privat by Design

Was ist ein Gantt-Diagramm? →
Ist das wirklich kostenlos?

Ja. Keine Konten, keine Testphase, keine kostenpflichtigen Funktionen. Alles läuft im Browser, es gibt also keine Server zu finanzieren.

Muss ich ein Konto anlegen?

Nein. App öffnen und loslegen. Ihr Projekt wird automatisch im Browser gespeichert.

Wo werden meine Daten gespeichert?

In Ihrem eigenen Browser (localStorage). Nichts wird auf einen Server hochgeladen. Sie können zusätzlich eine .gantts-Datei auf Ihrem Rechner speichern.

Kann ich nach Excel oder PowerPoint exportieren?

Ja, ebenso nach PDF, PNG und CSV. Die Exporte sind echte, bearbeitbare Dateien — keine Bilder.

Bereit, Ihr Projekt zu planen?

Ohne Anmeldung, ohne Kreditkarte. App öffnen und anfangen.
