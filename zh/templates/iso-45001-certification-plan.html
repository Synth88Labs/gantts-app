ISO 45001 认证计划模板

一份免费的 ISO 45001 认证计划模板（职业健康安全管理体系），把员工协商与参与作为有日期的排期工作画在图上——因为这是标准的一项要求，不是宣传上的锦上添花。另一条脊柱是危险源辨识与风险评价：目标指标、运行控制和培训全都从它推导出来。审核机制与 ISO 9001 认证计划、ISO 14001 认证计划和 ISO 27001 认证计划是同样的两阶段形状。需要说明的是：ISO 45001 是一张自愿性的第三方认证，它不替代《安全生产法》为你规定的任何法定义务。

模板包含什么

注意协商那根条几乎贯穿整张图，而事故与未遂事件记录那个阶段压不动。两者都是有意为之：

  • 范围、环境与差距分析 — 职业健康安全管理体系的边界，员工与其他相关方的识别，对照标准做差距分析，职业健康安全方针，以及认证机构选择——两次审核的日期就是在这时候定下来的。选机构时确认它经 CNAS 认可，并且认可范围覆盖你的行业。里程碑：范围与方针批准。
  • 危险源辨识与风险评价 — 整个体系的发动机：梳理作业活动与工序，辨识常规与非常规作业中的危险源，按消除、替代、工程控制、管理控制、个体防护的控制层级选取措施，同期建立法律法规与其他要求清单——在国内，这份清单的底座是《安全生产法》及配套的行业标准。里程碑：危险源与风险清单批准。
  • 员工协商与参与 — 排期，而不是默认发生。选举员工代表，成立有议事规则和固定会议周期的安全生产委员会，就危险源工作和目标草案征求意见，并专门检视一下是什么在妨碍大家真正参与。里程碑：协商机制运转起来。
  • 目标、运行控制与应急准备 — 由已评价风险导出的目标指标，带指标值和责任人；高风险作业的运行控制，作业票与相关方（承包商）管理；三级安全教育与岗位能力培训；以及应急预案编制、按规定向应急管理部门备案，并组织演练。里程碑：运行控制到位、演练完成。
  • 体系运行、事故与未遂事件记录 — 压不短的那个阶段：让体系真跑足够久，久到确实存在可供抽样的事故报告、未遂事件记录、调查、已闭环的纠正措施和一次合规性评价。里程碑：事故与未遂事件记录可供抽样。
  • 内审、管理评审与两阶段审核 — 内审现场工作及其纠正措施，管理评审，然后是一阶段审核、纠正措施间隔期，以及二阶段审核。里程碑：ISO 45001 证书签发。
协商是 ISO 45001 与其他管理体系标准差别最大的地方，也是各家计划最常把它缩水成一张海报的地方。标准期望员工——包括非管理岗员工——被征求意见并参与到具体事项中：危险源辨识、风险评价、确定控制措施、事故调查。那意味着会议、纪要、代表和证据，所以它该带着日期出现在图上。第二个陷阱是记录生成期。二阶段审核抽的是事故报告、未遂事件报告、调查和已闭环措施；如果你的报告文化是在审核前三周才开始的，就没有东西可抽，也看不出趋势。早点开始未遂事件上报，并把未遂事件数量上升当成成功，而不是失败。还要讲清一件事：这张证书不替代法定安全生产责任。安全生产标准化达标、应急预案备案、特种作业人员持证、三级安全教育、隐患排查治理这些都是法定的，认证做得再漂亮也不豁免；反过来，法定项做到位也不会自动换来证书。

如何调整

  1. 早点定认证机构并把二阶段审核日期钉住；它前面的一切就是倒计时，而不是估算。
  2. 把危险源辨识那几行展开成每个厂区、每个部门或每项高风险作业一行——一行掩盖真实工作量。
  3. 委员会的会议周期按你自己的节奏设，并让这根条一直跑过二阶段审核；审核员看的是持续性，不是一阵突击。
  4. 一阶段与二阶段之间至少留六周，好让一阶段提出的纠正措施真的关得掉。
  5. 如果你的事故数量很低，就把体系运行期拉长——记录要多到能看出趋势才行。
  6. 把落在这个窗口内的安全生产标准化评审、法定检验检测、职业健康体检轮次或监管部门检查也加成行，它们抢的是同一批人。

排期建议

  • 协商要排期，不要假定它会发生。有名有姓的员工代表、固定的会议周期和纪要才是证据；一个内网页面不是。
  • 非常规作业要覆盖到。检维修、停车大修、承包商作业和应急处置，是危险源清单最明显单薄的地方。
  • 明确使用控制层级。审核员会问你为什么选了个体防护而不是消除或替代，并且期待一个有记录的回答。
  • 尽早鼓励未遂事件上报。未遂事件正是二阶段用来证明体系在被使用的那批记录，而它们要老老实实地积累几个月才够。
  • 把四张 ISO 证书区分清楚。ISO 45001、ISO 9001、ISO 14001 和 ISO 27001 共用高阶结构、可以合并审核，但它们是四张彼此独立的认可认证，内容各不相同。
  • 别拿认证去顶法定义务。《安全生产法》下的责任、应急预案备案、特种作业持证和标准化达标，都要各自完成；证书证明的是体系在运转，不是合法性结论。

常见问题

ISO 45001 认证要多久？

从零起步常见十到十六个月，模板用了大约十七个月。压不动的两段，一是前面的危险源与风险评价工作，二是后面那个为审核员生成事故与未遂事件记录的运行期。

为什么员工协商要排成真实工作？

因为标准要求它。ISO 45001 要求组织就危险源辨识、风险评价、确定控制措施、事故调查等事项征求员工（包括非管理岗员工）的意见并使其参与，还要消除参与的障碍。这意味着代表、委员会、会议周期和记录，全都有日期，所以模板给它们排的是条，不是一句宣传语。

ISO 45001 和 ISO 9001、ISO 14001、ISO 27001 有什么不同？

它们共用同一套管理体系高阶结构和同样的两阶段审核机制，但内容完全不同。ISO 45001 是职业健康安全，ISO 9001 是质量，ISO 14001 是环境，ISO 27001 是信息安全。四者是四张独立的认可认证，各有范围和证书，为其中一张准备的证据不能满足另一张。另外要注意，SOC 2 是 AICPA 的鉴证报告，根本不是 ISO 标准——如果你要的是它，请看 SOC 2 合规时间表。

ISO 45001 能替代安全生产标准化吗？

不能，两者是不同的东西。安全生产标准化是国内依据《安全生产法》体系推行的评审定级，ISO 45001 是自愿性的第三方认证。两者在危险源辨识、隐患排查、教育培训和应急管理上有大量重叠，证据可以互相复用，但达标定级不等于拿到证书，拿到证书也不豁免任何法定义务。

能不能做几张标准的一体化管理体系？

可以。一体化管理体系的意思是把共用条款——方针、组织环境、内审、管理评审、成文信息——运行一次而不是四次，认证机构也常常合并成一次审核。但它不合并证书：你仍然持有针对不同标准、不同范围的独立证书，标准特有的内容也仍然各自独立。

二阶段审核之前需要多少条事故记录？

没有固定数量，审核员抽的是"这个流程在被使用、发现的问题走向闭环"的证据。一条未遂事件报告都没有的体系，读起来是上报出了问题，而不是现场特别安全。请在二阶段之前把上报跑上几个月，让趋势、调查和已验证的纠正措施都拿得出来。
