免费甘特图软件：哪些是真的免费

在项目管理软件里，"免费"有三种含义：长期免费、限额内免费、或者试用期免费。这个差别通常要等项目已经跑起来才会显现。

三种免费

真正免费：功能完整、不限人数、没有期限。少见，主要出现在没有服务器成本的工具上：浏览器里计算的网页工具，或本机安装的开源软件。

增值收费（Freemium）：限额之内长期免费，但总会卡住某样东西——一个项目、约六十项任务、三到五个用户，或者把导出和关键路径放进付费档。超出就要付费，而那时数据已经在里面了。

试用期：14 到 30 天内全部功能开放，到期必须付费。这不是免费软件，这是演示。

三种模式都正当，问题只在于后两种也常被笼统写成"免费"。

主要工具逐个看

  • gantts.app——我们自己的工具。浏览器里运行，不注册、不限任务和项目数，自动关键路径、依赖关系和里程碑，一键导出 PDF、PNG、Excel、PowerPoint，数据留在本地。代价是没有工时统计和资源计费功能。
  • GanttProject——免费开源桌面软件，支持依赖关系、关键路径、资源分配，可导出 PDF、PNG、CSV，离线且不用账号，但界面偏旧、没有云同步。
  • TeamGantt——协作出色，免费档通常只有一个项目、约六十项任务，关键路径和完整导出在付费档，必须注册。
  • GanttPRO——设计精良的商业工具，提供试用而非免费档，到期即付费。
  • Instagantt——以 Asana 集成见长，增值收费，进阶导出和关键路径在付费档。
  • Canva——图做得好看，免费档可导出 PNG 和 PDF，但它是设计工具：没有依赖关系，也没有关键路径，横条只是手动摆放的图形。
  • Microsoft Project——排程能力最强，资源平衡、基准、关键路径俱全，但没有免费档。
  • Excel 与 Google 表格——用堆积条形图或「时间轴」视图能凑出甘特图，但没有依赖关系和关键路径，全靠手工维护。

各家免费额度会变，动手前请到官网价格页确认。

选型时确认什么

四个问题基本能决定一切。它会计算依赖关系吗？不会的话，它只是个画图程序。能显示关键路径吗？——这一项在商业工具里最常被放进付费档。数据能不能导出为 CSV、XLSX、PDF 或项目文件格式？数据存放在哪里？涉及人员和客户信息时，这不是小事。

依赖关系、关键路径、导出，恰好是免费档最容易让人失望的三处。这三项若重要，一开始就选免费提供它们的工具。

关于注册

多数免费工具在你看到第一条横条之前就要求注册。这在技术上几乎从来不是必需的：它是价格。完全在浏览器里运行、数据存在本地的工具既不需要账号也不需要服务器。

真正不用注册的选项其实很短：gantts.app（打开网页就能用，计划存在本机浏览器里），以及 GanttProject（不用账号，但要下载安装）。TeamGantt、GanttPRO、Instagantt、Canva 都要先建账号。

不上传也顺带回答了合规问题：没有离开设备的数据，不需要额外的数据处理协议约定。

怎么选

  • 只要快速做一张能导出的甘特图 → gantts.app，免注册，含关键路径与 Excel、PPT 导出。
  • 本地安装、离线、开源 → GanttProject。
  • 正在评估付费协作平台、用量在限额内 → TeamGantt 或 Instagantt 免费档。
  • 企业级项目群管理、需要资源平衡 → Microsoft Project 或 GanttPRO，均为付费。
  • 只需要一张放进幻灯片的静态图 → Canva，或从 gantts.app 导出 PNG。

什么时候值得付费

多人同时编辑同一份计划时；需要跨项目做资源规划和资源平衡时；或者对甲方有正式报告义务、需要基准与偏差留痕时。三十项任务的独立计划，付费软件几乎总是过度配置。

开始之前先确认导出功能。数据无法以开放格式导出的工具，无论价格多少都是昂贵的——更换工具的代价，就是把整份计划重做一遍。

常见问题

哪些甘特图软件是真正免费的？

gantts.app 完全免费且不需要账号，GanttProject 是免费开源的桌面软件。TeamGantt 和 Instagantt 提供有限额的免费档，GanttPRO 只有试用，Microsoft Project 完全付费。

免费、增值收费和试用有什么区别？

免费是无成本也无期限；增值收费长期免费但限制额度或把关键功能放进付费档；试用是短期内全功能开放，到期必须付费。很多标着"免费"的工具其实是后两种。

有不用注册的免费甘特图工具吗？

有。gantts.app 打开网页就能用，不需要账号，计划保存在你自己的浏览器里。GanttProject 也不需要账号，但要下载安装。多数网页平台都要求先注册。

哪些免费工具带关键路径？

gantts.app 免费提供自动关键路径，GanttProject 在桌面端也支持。不少商业工具把关键路径留在付费档，动手前值得确认。

选型时应该看什么？

依赖关系、关键路径、导出格式，以及数据存放位置。其余多属于使用体验层面，可以之后再权衡。

免费软件适合企业使用吗？

独立的计划完全适合。多人协同编辑、跨项目资源规划和正式的基准偏差报告，通常需要付费工具。
