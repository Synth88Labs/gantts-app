What's included

The template is organized into the five phases of a typical build, each pre-filled with common tasks and milestones so you can adapt rather than start from a blank page:

  • Pre-construction — site survey, permits and approvals, design finalization, procurement and contractor selection. Milestone: permits approved.
  • Foundation — site clearing and excavation, utilities and drainage, footings, foundation pour and curing. Milestone: foundation complete.
  • Structure — framing, roofing, exterior walls, windows and weatherproofing. Milestone: building watertight / dry-in.
  • Interior fit-out — rough-in (electrical, plumbing, HVAC), insulation, drywall, flooring, fixtures and finishes. Milestone: inspections passed.
  • Handover — final inspection, snagging / punch list, cleaning, client walkthrough and handover. Milestone: project handover.
Dependencies matter most on site. Framing can't start before the foundation cures; drywall waits on rough-in inspection. Open this in the free gantts.app editor to link those relationships — when one trade slips, downstream tasks shift automatically and the critical path updates. No sign up.

How to customize it

  1. Set the project start date and let the phases cascade from there.
  2. Adjust durations to your scope — a single-family home and a commercial fit-out have very different timelines.
  3. Add trade-specific tasks under each phase (e.g. tiling, landscaping, elevator install).
  4. Assign responsibility in the Owner column — GC, subcontractor, or client.
  5. Link dependencies and mark inspection gates as milestones so nothing proceeds out of sequence.

Construction scheduling tips

  • Build in weather and float. Exterior and foundation work is weather-sensitive; add buffer days rather than assuming a perfect run. The editor's working-day calendar excludes weekends automatically.
  • Watch the critical path. The chain running through foundation → structure → dry-in usually drives the finish date. Protect those tasks; a slip there slips the whole job.
  • Sequence inspections as milestones. Treat each inspection or sign-off as a hard gate so trades don't start work that has to be re-done.
  • Coordinate long-lead procurement. Order items with long lead times (windows, steel, custom joinery) during pre-construction so they don't stall the build later.
  • Update % complete weekly and re-export for site meetings so everyone works from the same schedule.
