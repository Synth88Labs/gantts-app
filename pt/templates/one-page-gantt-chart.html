Gráfico de Gantt em uma página

Um gráfico de Gantt em uma página não é um cronograma reduzido: é um cronograma resumido. Mostra fases e marcos e deixa o detalhe no cronograma de trabalho — porque uma página não comporta os dois.

⬇ Excel (.xlsx) ⬇ PowerPoint ⬇ CSV ✎ Editar online grátis

O que o modelo contém

O que a página contém:

  • Fases — De quatro a oito barras cobrindo todo o projeto.
  • Marcos — De cinco a dez losangos para decisões e entregas.
  • Eixo — Graduação mensal ou trimestral conforme a duração.
  • Situação — Uma marca do dia atual e, se útil, um indicador de avanço por fase.
Uma página obriga a escolher, e é justamente esse o valor. Se você não consegue reduzir o projeto a oito barras, normalmente não é problema de diagramação: é um projeto cuja estrutura ainda não foi definida.

Como adaptar

  1. Junte fases vizinhas se o cronograma passar de oito barras.
  2. Mantenha apenas os marcos que um comitê guardaria.
  3. Acrescente uma linha de situação por fase se a página servir para acompanhamento.
  4. Exporte em PDF ou imagem para inserir em um documento.

Dicas de cronograma

  • De quatro a oito fases no máximo. Acima disso, a página perde o sentido.
  • Deixe o detalhe em outro lugar. Esta página remete ao cronograma, não o substitui.
  • Date a versão. Uma página resumida circula e vence rápido.
  • Uma marca de "hoje" vale mais que um percentual de avanço.

Perguntas frequentes

Para que serve um Gantt em uma página?

Para apresentar o projeto a uma diretoria ou comitê e servir de referência comum. O acompanhamento se faz no cronograma detalhado.

Quantas fases cabem?

De quatro a oito. Se o projeto exigir mais, agrupe antes de reduzir o tamanho da fonte.

Como manter atualizada?

Gerando-a novamente a partir do cronograma detalhado, em vez de editá-la à parte.
