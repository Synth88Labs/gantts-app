How to Make a Gantt Chart in Excel

By the gantts.app teamUpdated July 14, 202610 min read

Excel has no built-in Gantt chart, but you can make one from a stacked bar chart: build a table of tasks with start dates and durations, chart it as a stacked bar, hide the start series, reverse the task order, then format the dates and colors. Does Excel have a Gantt chart?

No — Excel has no native Gantt chart type. If you open the Insert Chart menu you will find columns, bars, lines, and pies, but nothing labeled "Gantt." The universally used workaround is to take a stacked bar chart and make the first stacked segment invisible, so the visible segments float at their start dates and line up exactly like Gantt bars. Microsoft also ships a few pre-built "Gantt project planner" templates you can download from the New workbook screen, but those are just spreadsheets using the same trick (or conditional formatting) under the hood. The steps below build one from scratch so you understand every part.

Step 1: Build a table with task, start, end & duration

On a blank sheet, create four columns: Task, Start, End, and Duration. Enter your tasks and real dates in Start and End, formatted as dates. Then compute Duration in days with a simple formula — if Start is in B2 and End is in C2, put =C2-B2 in D2 and fill it down. Excel stores dates as serial numbers, so the subtraction returns a plain number of days, which is exactly what the chart needs. A small example:

TaskStartEndDuration
ResearchJul 1Jul 54
DesignJul 5Jul 127
BuildJul 12Jul 2614
TestJul 26Aug 27
LaunchAug 2Aug 42

Step 2: Insert a stacked bar chart

Select the Task and Start columns together, then hold Ctrl and also select the Duration column so all three are highlighted (leave End out — the chart only needs Start and Duration). Go to Insert → Charts → Bar → 2-D Stacked Bar. Excel draws a chart with two series stacked in each row: a Start segment and a Duration segment. It will not look like a Gantt chart yet — that is expected. If Excel picks up the wrong data, use Select Data to confirm Start is series one and Duration is series two.

Step 3: Make the Start series No Fill

This is the key move. Click once on any Start segment to select the whole Start series, right-click and choose Format Data Series, open the paint-bucket (Fill) options, and set Fill: No Fill. The Start segments vanish, leaving each Duration bar floating at its correct start date. Suddenly it reads like a Gantt chart. If you also see a border, set Border: No Line so no outline remains where the invisible segment used to be.

Step 4: Reverse the task order

Excel plots the first table row at the bottom of a bar chart, so your tasks appear upside down. Fix it by clicking the vertical axis (the list of task names), right-click, choose Format Axis, and under Axis Options tick Categories in reverse order. Your first task jumps to the top and the list now reads top-to-bottom in the same order as your table. You will usually also want to tick "Horizontal axis crosses: At maximum category" in the same panel so the date axis stays at the top.

Step 5: Format dates & colors

Two finishing touches make it look professional. First, the date axis usually starts at zero, leaving a big empty gap. Click the horizontal axis, open Format Axis, and set the Minimum bound to your project's start date and the Maximum to its end date. Excel wants these as serial numbers — the quickest way is to type the date in an empty cell, note the number it becomes when formatted as General, and enter that. Then set a sensible major unit (for example 7 for weekly gridlines) and a clean date number format. Finally, click the Duration series and recolor the bars, add data labels if you like, and delete the legend. You now have a working Excel Gantt chart.

Adding dependencies & % complete

Excel's stacked-bar approach has real limits, and this is where they show. Dependencies are not supported natively — there is no way to draw finish-to-start arrows, and moving one task will not shift the tasks that depend on it. The common workaround is to make each task's Start a formula referencing the previous task's End (for example =C2), so a change ripples down the column, but you still get no visible arrows and only simple chains.

Percent complete can be faked with a helper column. Add a % Complete column, then a Progress helper column equal to Duration × % Complete. Add that Progress helper as a third stacked series in a darker shade so a filled portion of each bar shows how far along the task is. It works, but it is fiddly to maintain — every new task means updating the chart's data range and re-coloring series.

Automatic Gantt via conditional formatting

If you would rather avoid charts entirely, there is a second method: a conditional-formatting Gantt built right in the cells. Lay out a calendar across the top row (one column per day or week), keep your task, start, and duration columns on the left, then select the calendar grid and add a conditional-formatting rule with a formula like =AND(dateInColumn>=$B2, dateInColumn<$C2) that fills a cell when its date falls within a task's date range. The result is a "heat-map" Gantt that updates automatically as you change dates — no chart object required. It prints cleanly and is easy to extend, though colors, milestones, and progress still take manual formula work.

Frequently asked questions

Does Excel have a built-in Gantt chart?

No. Excel has no native Gantt chart type. The standard method is to start from a stacked bar chart and hide the first series so the remaining bars line up like a Gantt chart. Excel also ships some Gantt project templates you can download.

How do I turn a stacked bar chart into a Gantt chart in Excel?

Chart the Start and Duration columns as a stacked bar, then select the Start series and set its fill to No Fill. The invisible Start segments push each Duration bar to its correct start date, producing a Gantt chart. Reverse the axis so task one sits at the top.

How do I add percent complete to an Excel Gantt chart?

Add a Percent complete column and a helper column that multiplies duration by percent complete. Add that helper as a third stacked series in a darker shade so a filled portion of each bar shows how much of the task is done.

Why is my Excel Gantt chart in the wrong order?

By default Excel plots the first table row at the bottom of a bar chart. Select the vertical axis, open Format Axis, and tick Categories in reverse order so your tasks read top to bottom in the same order as your table.
