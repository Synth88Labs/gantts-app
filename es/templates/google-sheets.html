Plantilla de diagrama de Gantt para Google Sheets

Una plantilla de diagrama de Gantt para Google Sheets en formato CSV, lista para importar y usar. Sirve tanto para la función Cronograma como para montar un gráfico de barras apiladas.

⬇ Excel (.xlsx) ⬇ PowerPoint ⬇ CSV ✎ Editar online gratis

Qué incluye

Los datos en el formato que Sheets necesita:

  • Columnas de datos — Tarea, inicio, fin, duración, avance y responsable.
  • Fases de ejemplo — Estructura de fases y subtareas para adaptar.
  • Hitos — Filas de duración cero para entregas y aprobaciones.
  • Compatible con Cronograma — Las columnas coinciden con lo que espera Insertar → Cronograma.
Google Sheets es excelente para colaborar sobre datos y limitado para planificar: no hay dependencias, ni ruta crítica, ni días laborables. Para un plan de veinte tareas se aguanta; para sesenta, cada cambio obliga a recalcular a mano.

Cómo personalizarla

  1. Importa el CSV en una hoja nueva y sustituye las filas de ejemplo.
  2. Usa Insertar → Cronograma para la vista rápida, o monta un gráfico de barras apiladas si quieres más control.
  3. Añade formato condicional por fase para distinguir bloques de un vistazo.
  4. Comparte con permiso de comentario en lugar de edición si el plan ya está acordado.

Consejos de programación

  • Comprueba el formato de fecha tras importar: Sheets a veces interpreta día y mes al revés según la configuración regional.
  • Congela la primera fila para no perder los encabezados al desplazarte.
  • Si necesitas dependencias, exporta el CSV al editor gratuito y enlázalas allí.
  • Duplica la hoja antes de reestructurar: el historial de versiones ayuda, pero una copia es más rápida.

Preguntas frecuentes

¿Cómo importo la plantilla en Google Sheets?

Descarga el CSV y usa Archivo → Importar en una hoja nueva. Las columnas ya coinciden con lo que espera la función Cronograma.

¿Sheets tiene diagramas de Gantt nativos?

Tiene una función de Cronograma que muestra barras sobre una línea de tiempo, pero sin dependencias ni ruta crítica.

¿Es gratuita?

Sí, sin registro.
